ВВЕДЕНИЕ.
----------------------

	Модуль Presentation Layer предназначен для обеспечения
	доступа сторонних приложений к устройствам парковки.

ЗАВИСИМОСТИ.
----------------------

	Указаны версии, с которыми проходила тестовая сборка

	boost-1.54
	cppunit-1.13
	
	Для сборки требуется подключить библиотеки
	boost_thread
	boost_system
	cppunit
	pthread


ИНСТРУМЕНТЫ.
-----------------------

	Исходный код сопровождается настройками проекта для
	IDE Eclipse. Текущая сборка и тестирование проводятся
	на версии:
		Eclipse IDE for C/C++ Developers
		Version: Neon Milestone 4 (4.6.0M4)
		Build id: 20151218-1739

	Eclipse автоматически создает makefile из настроек проекта.
	
ТЕСТИРОВАНИЕ.
-----------------------

	Исходный код для тестирования библиотек содержится в проекте
	UnitTests. В данный момент проект содержит только тесты для
	TcpIp Server и имеет соотвествующее название testTcpIpMultiServer.
	В дальнейшем проект 

	TcpIp Server
	содержит 2 теста, повторяющихся 5000 раз в цикле. Если эта библиотека
	будет запланирована на продакшн после пилотника, то список тестов
	надо будет расширить тестами непрерывной приемо-передачи, тестами 
	на переполнение буфера и тестами автореконнектов.

	Presentation Layer
	На текущий момент тестов не имеет.
